## v0.5.0 Release Notes

- Add support to export and import schedules in JSON format, which can be used for backup and restore or transferring schedules between devices.
- Add support to crawl the course schedule for Huazhong University of Science and Technology.
- Add support to manually clean the cookies for the course schedule crawler, which can be used to fix the login issues.
- Add support to set the date format for the calendar.
- Add support to set the theme mode to follow system, light or dark.

---

## v0.5.0 版本更新日志

- 添加了对于使用 JSON 格式导出和导入课表的支持，可以用于备份和恢复或在设备之间传输课表。
- 添加了对华中科技大学课程表爬取的支持。
- 添加了手动清除课程表爬虫的 cookies 的支持，可以用来修复登录问题。
- 添加了设置日历日期格式的支持。
- 添加了设置主题模式为跟随系统、亮色或深色的支持。

## v0.5.0 版本更新記錄

- 新增了對使用 JSON 格式匯出和匯入課表的支持，可以用於備份和恢復或在流动設備之間傳輸課表。

## v0.5.0 リリースノート

- JSON 形式でのスケジュールのエクスポートとインポートのサポートを追加しました。これを使用して、スケジュールのバックアップと復元や、デバイス間でのスケジュールの転送ができます。
- 華中科技大学のコーススケジュールをクロールするサポートを追加しました。
- コーススケジュールクロールのクッキーを手動でクリアするサポートを追加しました。これを使用してログインの問題を修正できます。
- カレンダーの日付形式を設定するサポートを追加しました。
- テーマモードをシステムに従う、ライト、ダークに設定するサポートを追加しました。
